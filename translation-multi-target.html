Image Generation Capabilities

SORA / CHATGPT

Added: Platform constraints
  • DALL-E generates at fixed resolutions (1024x1024, 1024x1792, 1792x1024). Use 1024x1792 for 3:4 portrait orientation (closest match).
  • DALL-E does not support negative prompts ("NO shading"). Rephrase as positive instructions: "flat solid color fills only, hard edges only."
  • DALL-E tends toward photorealism by default. Explicitly lead with "simplified graphic illustration" to override.
  • DALL-E handles hex colors better than Gemini. Include hex codes confidently.
ADD_PLATFORM_CONSTRAINTS — DALL-E's fixed aspect ratios, lack of negative prompts, and photorealism bias are critical to getting the right output. Without these notes, the model would generate wrong-ratio photorealistic images.

MIDJOURNEY

Added: Parameter syntax
  • Aspect ratio: --ar 3:4
  • Style raw (less "Midjourney look"): --style raw
  • No text in image: --no text labels watermarks
  • Stylization (lower = more literal): --stylize 200
  • Version: --v 7
ADD_PLATFORM_SYNTAX — Midjourney uses CLI-style parameters appended to prompts. These don't exist on any other platform. A user unfamiliar with MJ would never know to add --style raw.

Safety / Content Policy

SORA / CHATGPT

Canonical: "Avoid naming specific historical figures directly"

OpenAI's content policy is stricter than Gemini's for historical figures. DALL-E will refuse prompts depicting real historical figures in stylized/caricatured form. Always use descriptive titles ("18th-century French military commander") and never proper names. If the user requests a specific figure, generate the description without the name.

ADAPT_SAFETY_POLICY — Each platform has different content policy boundaries. The canonical note is generic; each target needs platform-specific guidance on what will and won't be blocked.

MIDJOURNEY

Midjourney is more permissive with historical figure names but will block prompts it interprets as depicting real people in harmful contexts. Generally safe to use names like "Julius Caesar" or "Cleopatra" for stylized illustration. Avoid combining names with violent or sexual language.

ADAPT_SAFETY_POLICY — Midjourney's content policy differs substantially from OpenAI/Google. What gets blocked on one platform sails through on another.

Color Accuracy

SORA / CHATGPT

Canonical: "Image generation does NOT reliably reproduce specific hex colors"

DALL-E has better hex color fidelity than Gemini. Include hex codes directly in prompts: "skin filled with muted yellow-green olive (#6B7F5E)". Results will be approximate but closer than Gemini. For exact colors, apply post-processing.

MIDJOURNEY

Midjourney does not interpret hex codes in prompts. Use only English color descriptions: "muted yellow-green olive skin" not "#6B7F5E". For palette control, use --sref (style reference) with a previously generated on-palette image as anchor.

ADAPT_COLOR_HANDLING — Midjourney's --sref parameter is a unique capability for style consistency that doesn't exist on other platforms. This is an example of a platform-specific upgrade, like Gemini's native image gen.

REFORMAT_PROMPT_SYNTAX — Midjourney uses a fundamentally different prompt grammar:
  • Multi-prompt weighting with ::N syntax to prioritize elements (style anchored at ::2, subject at ::1)
  • Comma-separated keywords instead of full sentences (MJ responds better to keywords)
  • --no parameter for negative prompts instead of inline "NO shading"
  • No hex codes — English color names only
  • Lowercase convention — MJ is not case-sensitive; lowercase is community convention
This is not a find-and-replace. It's a structural rewrite driven by how the model interprets input.

The key insight

The art direction knowledge (palettes, moods, style references, composition rules) transfers at nearly 100% across all platforms. That's the valuable, hard-won intellectual property.

What changes per platform is the delivery mechanism: how you express that knowledge in a way the target model/tool can act on. For conversational models (Gemini, ChatGPT), it's a system instruction. For Midjourney, it's a prompt construction reference with platform-specific syntax.
